# Dag Motorsports

Dag motorsports is a portal that allows users to submit & compare lap times set in various racing games. Lap times are tracked by user, game, track & car used.

## Main gems used
- Rails
- Devise (for user accounts)
- RSpec (for tests)
- HAML (for frontend)

## Games supported
- Forza Motorsport 7
- Assetto Corsa
- Gran Turismo Sport
- Automobilista
